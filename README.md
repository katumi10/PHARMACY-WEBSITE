
# PHARMACY-WEBSITE
# 💊 PharmaGH – Online Pharmacy Website (Frontend)

PharmaGH is a clean and responsive frontend for a Ghana-based online pharmacy.
This project focuses on building a modern, user-friendly homepage using **HTML, CSS, and minimal JavaScript**.

---

## 🚀 Features

* 🟢 Clean and modern UI design
* 📱 Fully responsive (mobile, tablet, desktop)
* 🔍 Search bar for products
* 🛒 Cart icon with item count
* 📦 Hero section with promotional content
* ✅ Trust indicators (Genuine Products, Licensed Pharmacy, Secure Payments)

---

## 🛠 Tech Stack

* **HTML5**
* **CSS3 (Flexbox + Media Queries)**
* **JavaScript (minimal)**
* **Font Awesome (icons)**

## 📱 Responsiveness

The layout adapts across different screen sizes using media queries:

* Mobile (≤ 480px)
* Tablet (≤ 768px)
* Small laptops (≤ 1024px)
